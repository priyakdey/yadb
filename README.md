# Yet Another Database



`yadb` is a 3AM idea. This is an **educational project** to learn 
- How a database works (sql database)
- How a reply works with a database
- How a parser works and grammar works
- How a database stores data in memory
- How a database flushes data to file
- When does a database flush data to a memory
- How and when does a datbase process loads data from the file to memory
